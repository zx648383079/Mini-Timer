# Mini-Timer
 小程序端时间管理

# 使用方法

配置 `src/config/config.ts` 复制`src/config/config.sample.ts` 内容 即可


## 编译单个文件

借助 Code Runner

在文件编辑框中右键 Run Code 即可编译当前文件 目前仅支持 .ts .vue

具体步骤请参考 【[博客](https://zodream.cn/blog/id/107.html)】
